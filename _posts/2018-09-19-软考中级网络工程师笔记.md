## 1 网络体系结构

### osi参考模型

|    层级    | 功能                                                       |
| :--------: | ---------------------------------------------------------- |
|   物理层   | 透明的比特流传输                                           |
| 数据链路层 | 物理链路加上通信协议形成的逻辑线路                         |
|   网络层   | 控制子网通信，提供路由选择                                 |
|   传输层   | 端到端数据传输，分段、传输、组装                           |
|   会话层   | 不同机器上用户建立会话                                     |
|   表示层   | 提供通用数据描述格式，不同系统间机器进行信息转换和相互操作 |
|   应用层   | 向应用程序提供服务                                         |
